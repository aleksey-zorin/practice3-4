В рамках цикла практических работ было разработано полнофункциональное Fullstack-приложение — интернет-магазин спортивных товаров. Проект включает серверную часть на Node.js/Express с REST API и клиентскую часть на React. Все работы (№3-5) выполнены и объединены в единую систему. Работы №1-2 были выполнены ранее и загружены отдельными коммитами.

Реализовано API с CRUD операциями для списка товаров:

| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | `/api/products` | Просмотр всех товаров |
| GET | `/api/products/:id` | Просмотр товара по id |
| POST | `/api/products` | Добавление товара |
| PATCH | `/api/products/:id` | Редактирование товара |
| DELETE | `/api/products/:id` | Удаление товара |
