В рамках цикла практических работ было разработано полнофункциональное Fullstack-приложение — интернет-магазин спортивных товаров. ### Практическая работа №2
[📄 Отчет фронтенд .docx](https://github.com/user-attachments/files/25678197/default.docx)

Реализовано API с CRUD операциями для списка товаров:

| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | `/api/products` | Просмотр всех товаров |
| GET | `/api/products/:id` | Просмотр товара по id |
| POST | `/api/products` | Добавление товара |
| PATCH | `/api/products/:id` | Редактирование товара |
| DELETE | `/api/products/:id` | Удаление товара |

**Структура товара:**
```json
{
  "id": "1",
  "название": "Футбольный мяч",
  "стоимость": 2500
}
----
